# SpritePacker ✨

![Client-Side Processing](https://img.shields.io/badge/Processing-100%25%20Client--Side-brightgreen)
![No Backend](https://img.shields.io/badge/Backend-None%20Required-blue)
![Tiled Compatible](https://img.shields.io/badge/Tiled-Compatible-orange)

SpritePacker is a high-performance, privacy-first Pixel Art Asset Pipeline and Sprite Sheet Generator built specifically for 2D game developers. It runs entirely in your browser using the HTML5 Canvas API, meaning your assets never leave your machine.

Designed to streamline workflows for grid-based games (like Tycoon or RPG genres), it automates the tedious processes of background removal, precise scaling, texture packing, and Tiled map editor integration.

## 🚀 Features

* **100% Client-Side Processing:** Lightning-fast image manipulation using your own CPU/RAM. No servers, no uploads, zero privacy concerns.
* **Smart Color Keying:** Automatically strips pure white (`#FFFFFF`) backgrounds generated by AI tools. Features an adjustable **Tolerance Threshold** to clean up noisy anti-aliasing pixels without damaging your core sprite.
* **Auto-Trim & Custom Padding:** Automatically crops empty space around your sprites and applies user-defined `X/Y Padding` before scaling them into their grid cells.
* **Grid-Aware Texture Packing:** Define how many grid blocks each asset should occupy (e.g., a chair is `1x1`, a large desk is `2x4`). The 2D Bin Packing algorithm handles the rest.
* **Hover Outline Generator:** Need selection/hover effects for your game engine? SpritePacker can generate and export a separate `_outlines.png` file with perfectly traced 1px borders.
* **Instant Tiled Integration:** Automatically generates and downloads a `.tsx` (Tileset XML) file alongside your sprite sheet, ready to be dropped straight into the Tiled Map Editor.
* **Pixel-Perfect Scaling:** Uses `Nearest Neighbor` interpolation to guarantee your 8-bit/16-bit pixel art remains perfectly sharp.

## 🛠️ How to Use

1.  **Set Base Resolution:** Choose your game's base tile size (16x16, 32x32, 48x48, 64x64).
2.  **Drag & Drop:** Upload your raw AI-generated or hand-drawn asset files.
3.  **Configure Spans & Padding:** Tell the packer how many grid cells each item takes (`WxH`) and set internal padding margins.
4.  **Tweak Tolerance:** If using the Background Removal tool, adjust the tolerance slider for a clean cut.
5.  **Generate:** Click export to instantly receive your packed `.png` sheet, your `.tsx` Tiled configuration, and your `_outlines.png` (optional).

## 💻 Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Processing:** HTML5 `<canvas>`, Vanilla JavaScript
* **File Handling:** `react-dropzone`
* **Deployment:** GitHub Pages (via GitHub Actions)
